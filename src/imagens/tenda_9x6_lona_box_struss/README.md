# Imagens da Tenda Box Truss 9x6

As imagens desta pasta são usadas no produto **Tenda Box Truss 9x6** (id: `tenda_box_truss_9x6`) em:

- Página Box (`/box`) – card do produto
- Página do produto (`/produto/tenda_box_truss_9x6`) – galeria

O mapeamento entre a pasta e o produto está em `src/utils/imagens.js` (productIdToFolder).
