# Imagens do Pórtico 6x4 (Pórtico de Entrada)

As imagens desta pasta são usadas automaticamente para o produto **Pórtico de Entrada 6m x 4,6m** em:

- **Página Box** (`/box`) – hero e card do pórtico
- **Página do produto** (`/produto/portico_de_entrada`) – galeria/carrossel
- **Home** – seção "Veja o que temos"

**Formatos suportados:** PNG, JPG, JPEG, WEBP, SVG, GIF, BMP, HEIC.

Para converter HEIC ou outros formatos para PNG, execute na raiz do projeto:

```bash
npm run convert-heic
```

A primeira imagem (em ordem alfabética pelo nome do arquivo) é usada como imagem principal; as demais aparecem na página de detalhes do produto.
